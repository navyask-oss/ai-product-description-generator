# 30-Day Sticker Mule Prep Plan

## Week 1: Foundation

- Day 1: Clean GitHub profile, pin portfolio repos, add README screenshots and demo links.
- Day 2-3: Complete Tour of Go and build a basic Go REST API with PostgreSQL.
- Day 4-5: Complete TypeScript handbook and migrate an existing ML API concept to TypeScript.
- Day 6-7: Build a Vite React UI that calls the Go API.

## Week 2: Stack Depth

- Day 8-9: Learn GraphQL basics and add a gqlgen layer to the Go API.
- Day 10-11: Practice PostgreSQL schema design, migrations, joins, and indexes.
- Day 12-13: Build the Go backend for the AI Product Description Generator.
- Day 14: Prepare Dockerfile, Cloud Run deployment notes, and README documentation.

## Week 3: Mobile and Collaboration Tool

- Day 15-16: Build a small Expo todo app in TypeScript.
- Day 17-19: Build Sticker Collaboration Tool backend with Go, GraphQL, and PostgreSQL.
- Day 20-21: Build the collaboration frontend in React.

## Week 4: Polish and Apply

- Day 22-24: Build Order Tracker Mobile App in Expo and deploy backend.
- Day 25-26: Write three technical posts about Go, AI product copy, and TypeScript migration.
- Day 27-28: Update LinkedIn, resume, and portfolio.
- Day 29: Submit application with personalized cover letter and project links.
- Day 30: Follow up and engage with public company content.
